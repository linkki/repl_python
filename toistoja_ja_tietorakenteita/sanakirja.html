SANAKIRJA

Python Dictionary eli sanakirja

Eräs tärkeä ja paljon käytetty tietorakenne Pythonissa on dictionary, suomeksi sanakirja.

Sanakirja koostuu avaimista ( englanniksi key ) ja arvoista (englanniksi value). Sanakirjan avaimia voi ajatella hakusanoina, joilla saa haettua sanakirjasta avainta vastaavan arvon.

Tässä vaiheessa tiedetään, että Python lista luodaan käyttämällä hakasulkeita []. Sanakirjan voi luoda vastaavasti aaltosulkeilla {}.



Sanakirjan avain-arvo-parit erotetaan toisistaan pilkuilla samoin kuin listassa. Avain-arvo-pari merkitään käyttämällä kaksoispistettä: avain: arvo aivan kuten ylläolevan kuvan koodissa.
